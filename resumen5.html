Otras Funciones predefinidas en Python para listas:
count()=cuenta el numero de veces que se encuentra un elemento dentro de una lista 
split()=para hacer una lista a partir de un string segun paramento de separacion
len(items.split())=calcula la cantidad de palabras


En Python:
%contar las veces que hay
>>> numeros=[1,2,34,5,6,7,8,9,1]
>>> print(numeros.count(1))
2
>>> nombres=["santa","col","col","andres","jose"]
>>> print(nombres.count("col"))
2
>>> items="1,2,3,4,5,6"
>>> items.split(",")
['1', '2', '3', '4', '5', '6']
>>> items="mi mama me mima"
>>> items.split()
['mi', 'mama', 'me', 'mima']
>>> items
'mi mama me mima'
>>> len(items.split())
4
